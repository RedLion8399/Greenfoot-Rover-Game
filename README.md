# Greenfoot-Rover-Game

Ths project is a secound fork of the original greenfoot scenario mentioned in my [Improved-Greenfoot-World](https://github.comRedLion8399/Improved-Greenfoot-World) Reposiory. As well as that it belongs to my computer sience class in school.


## The Task
Erstelle ein Computerspiel mit der Planetenerkundung in Greenfoot.
Spielidee:

Es gibt zwei Rover, die sich bekämpfen. Sie werden mit der Tastatur gesteuert (ASDW, Pfeiltasten,…).
Rover können schießen (z.B. Granaten, Laser, …). Der Schuss bewegt sich geradlienig (links, rechts, oben, unten) über das Spielfeld, bis er auf den Rand, einen Rover oder einen Hügel trifft. 
Trifft ein Schuss einen Rover, so verliert dieser einen Lebenspunkt. Hat ein Rover alle Lebenspunkte verloren so verbrennt er, der Gegner gewinnt und das Spiel endet.

Die Munition ist begrenzt, es spawnen gelegentlich Gesteine, die eine zufällige und nicht erkennbare Anzahl an Munition enthalten.
Der Zustand der beiden Rover (Lebenspunkte, Munition) wird in einem Dashboard am Bildschirmrand angezeigt.
Relevante Spielereignisse (z.B. schießen, treffen, verlieren,…) sollen mit einer Soundausgabe unterlegt werden.
Bei Sounds und Graphiken dürfen frei verfügbare Bibliotheken genutzt werden.

Das Spiel soll für den Kurs spielfertig bereitgestellt werden. Die erstellte Architektur und Programmierung des Spiels soll in einem kurzen Vortrag vorgestellt werden.
